Yield Curve Comparisons
=======================

Purpose
-------

This page renders the accepted TSA29 TIPSY-vs-VDYP comparison library directly
from the checked-in ``plots/`` artifacts.

Use it when you want to inspect the current managed-AU treated-curve surface
without leaving the standalone docs.

How To Read These Figures
-------------------------

- treat each figure as one managed-AU treated-versus-natural comparison;
- focus first on broad shape, peak timing, and long-run divergence;
- use the figures as interpretation aids, not as substitutes for the canonical
  bundle, XML, or tracks surfaces; and
- use :doc:`figure-appendix` for the broader figure catalog and
  :doc:`rebuild-and-qa` for rebuild/evidence workflow context.

  :doc:`rebuild-and-qa`; and
