# fragments (externalized)

Patchworks fragment geometry artifacts are externalized for thin-repo delivery.

Use shared FEMIC data access workflow to retrieve the validated TSA29 fragment
bundle, then place files in this directory.

Expected filenames:

- `fragments.shp`
- `fragments.dbf`
- `fragments.shx`
- `fragments.prj`
- `fragments.cpg`

Reference hash for `fragments.shp` is recorded in
`metadata/large_artifacts.sha256`.
